Port-O-Burger

It is the home page of a (fictional) vegan burger restaurant. For this project I use HTML, CSS and a little bit of JS. The high quality images were compressed to improve page performance, it is adapted for any screen size.

Upcoming improvements:

  • -Some aesthetic details on the margin of the images.
  • -Reorganization of the sections to give focus to the menu.
  • -Reduction in the size of the images in the Ingredients section.
  • -Turn the menu into a "carousel" of elements.
  • -Improvements in Accessibility (Very important!).

Veterinaria San Michi

This is the main page of another fictional project: Veterinaria San Michi.
In this project I use HTML, CSS and some JS -the core of Web Development. Within the page you can schedule an appointment, which generates an alert in the window.

Upcoming improvements:

  • -Add more interactivity with JS.

  • -General improvements in terms of color.
  • -Improvements in Accessibility (Very important!).
  • -Significant improvements in performance.
